## 📌 О проекте

**Тема курсовой работы:**  
Разработка автоматизированной информационной системы для управления деятельностью типографии

**Цель работы:**  
Автоматизация процессов учёта заказов, расчёта стоимости печати и контроля загрузки оборудования в типографии.

**Актуальность:**  
Повышение эффективности работы типографии за счёт централизации данных и сокращения времени обработки заказов.

## 🎯 Основные возможности

| Функция | Описание |
|---------|----------|
| 📋 Учёт заказов | Регистрация, статусы, история выполнения |
| 💰 Расчёт стоимости | Автоматический расчёт с учётом тиража, формата, красочности |
| 📦 Учёт материалов | Контроль остатков бумаги, краски, плёнки |
| ⚙️ Загрузка оборудования | Планирование загрузки печатных машин |
| 📊 Отчёты | Загрузка цеха, прибыль, рентабельность заказов |
